# Data Science Project

## 📊 Overview
This project focuses on predicting startup success using machine learning.

## 📁 Files
- cleaned_startup(4).csv → Cleaned dataset
- uncleaned_startup_success_failure_5000.csv → Raw dataset
- preprocessing.ipynb → Data cleaning and preprocessing
- startup_visualization.ipynb → Data visualization
- modelfitting.ipynb → Model training and evaluation

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🎯 Model Performance
- Achieved accuracy: ~95%

## 🚀 How to Run
1. Open notebooks in Jupyter Notebook
2. Run preprocessing.ipynb
3. Run visualization
4. Run model fitting
